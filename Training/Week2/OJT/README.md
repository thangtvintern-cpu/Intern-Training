**Enviroment Setup**

1. Clone Project
- git clone <https://github.com/thangtvintern-cpu/Intern-Training>

2. Create Virtual Environment
- python -m venv .venv
- .venv\Scripts\activate

3. Install Pytest
- pip install pytest

4. Run Project
- python controller.py

5. Run Test
- pytest
